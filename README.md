# Experiment 03 — Python Loops & Conditionals

A set of five beginner Python programs practising `for`/`while` loops, conditionals, and user input.

## Problems

| File | Description |
|------|-------------|
| [problem01.py](problem01.py) | Print the multiplication table (1–10) for a user-supplied number |
| [problem02.py](problem02.py) | Calculate the sum of digits of a user-supplied integer |
| [problem03.py](problem03.py) | Check whether a user-supplied year is a leap year |
| [problem04.py](problem04.py) | Print a 5-row pattern of squared row numbers (`1, 4, 9, 16, 25`) |
| [problem05.py](problem05.py) | Print an increasing number triangle (row *n* shows `1 2 … n`) |

## Running a problem

```bash
python problem01.py
```

Each script that requires input will prompt you in the terminal.

## Requirements

- Python 3.x (no third-party packages; `calendar` is part of the standard library)
